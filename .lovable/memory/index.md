# Project Memory

## Core
Subulus-Salam: Islamic online school platform. Sage green/gold/cream palette. Amiri headings, Inter body.
Platform renamed to SABILUL JANNAH INTERNATIONAL ONLINE ISLAMIYYA. UI terminology: Course→Class, Module→Book.
Strict book progression: students must complete all lessons/quiz/assignment before next book unlocks.
Three roles: Student, Teacher, Admin. Mobile-first design.
DB tables keep original names (courses, modules) — only UI labels changed for backward compatibility.
