# Project Memory

## Core
Subulus-Salam: Islamic online school platform. Sage green/gold/cream palette. Amiri headings, Inter body.
Strict module progression: students must complete all lessons/quiz/assignment before next module unlocks.
Three roles: Student, Teacher, Admin. Mobile-first design.

## Memories
- [Design tokens](mem://design/tokens) — Islamic palette: sage green, gold, cream. Custom gradients and shadows.
- [Platform features](mem://features/platform) — Module lock system, AI assistant, referral, gamification, ads policy
- [Donations & Sponsorship](mem://features/donations) — Anonymous donations, campaigns, student sponsorship via Flutterwave
